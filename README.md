# 🌐 Maida Butt - Personal Portfolio

This is my **personal portfolio website** built with **HTML, CSS, and JavaScript**.

- 🎨 Fully responsive layout (desktop & mobile)  
- 🌀 Smooth scrolling and subtle animations
